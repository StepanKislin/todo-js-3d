# 3D TODO List

![3D TODO List Preview](https://via.placeholder.com/800x400/1e1b4b/ffffff?text=3D+Minimalist+TODO+App)

Современный минималистичный TODO-лист с интерактивным 3D-интерфейсом и многими списками задач. Все данные хранятся локально в браузере.

## ✨ Особенности

- **3D-анимация фона** с эффектом параллакса при движении мыши
- **Плавные анимации** добавления/удаления задач
- **Адаптивный дизайн** для всех устройств
- **Подтверждение критических действий** через модальные окна
- **Автоматический перенос текста** без горизонтальной прокрутки
- **Темная тема** с полупрозрачными элементами и размытием
- **Множество списков задач** с возможностью добавления, переименования и удаления
- **Перемещение окон** перетаскиванием за заголовок
- Хранение данных в **localStorage** (задачи сохраняются после перезагрузки)
- Кастомный **стилизованный скроллбар**

## 🚀 Как использовать

1. Скачайте все файлы проекта
2. Откройте `index.html` в современном браузере
3. Используйте кнопку "+" в левом верхнем углу для создания новых списков задач
4. Перетаскивайте окна за заголовок для изменения их положения
5. Двойной клик по заголовку позволяет переименовать список
6. Кнопка "×" в правом верхнем углу окна позволяет удалить список задач

## 📱 Адаптивность

Приложение корректно отображается на:
- Десктопах (1920px+)
- Планшетах (768px)
- Мобильных устройствах (320px+)

Особое внимание уделено:
- Автоматической высоте контейнера задач
- Размерам кнопок для тач-устройств
- Читаемости текста на маленьких экранах
- Оптимизации производительности 3D-рендеринга

## 🛠 Технологии

- **Three.js** (v128) - для 3D-графики
- **Vanilla JavaScript** - логика приложения
- **CSS3** - анимации и адаптивный дизайн
- **localStorage** - хранение данных
- **HTML5** - семантическая разметка
